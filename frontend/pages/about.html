Nepal WebGIS Project

Gantabya Sahayak
गन्तब्य सहायक

Gantabya Sahayak is a Smart Tourism WebGIS Platform designed to solve real geospatial challenges faced by tourists exploring Nepal. The platform integrates interactive web mapping, spatial analysis, real-time weather data, and a comprehensive database of Nepal's tourism resources.

The study area covers all of Nepal, with administrative boundaries at three levels: 7 Provinces, 77 Districts, and Local Level (Palikas). All features and modules reference these administrative units for spatial context.

Problem Statement: Tourists in Nepal face critical challenges: no real-time road condition updates, difficulty locating emergency services, lack of seasonal guidance, and no platform consolidating geospatial tourism intelligence. This platform addresses all of these through integrated WebGIS modules.

The frontend is a single-page-like application using standard HTML/JS that communicates with the FastAPI backend via REST API calls. All spatial data is returned as GeoJSON and rendered client-side using MapLibre GL JS. For offline/demo scenarios, the app falls back to local static GeoJSON files. The backend performs server-side spatial operations using GeoPandas and PostGIS for advanced queries.
